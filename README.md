# FUTURE_DS_02
📊 Customer Churn Analysis Dashboard (Power BI)
📌 Project Overview
This project presents an interactive **Customer Churn Analysis Dashboard** built using Power BI. 
The goal is to analyze customer behavior, identify key factors influencing churn, and provide actionable insights for improving customer retention.
📂 Dataset
* Source: Kaggle (Telco Customer Churn Dataset)
* Contains customer demographics, services, account information, and churn status
🎯 Objectives
* Analyze customer churn patterns
* Identify factors affecting customer retention
* Visualize trends in customer behavior and revenue
* Provide business insights for decision-making
📊 Dashboard Features
* 📈 Churn analysis by contract type and tech support
* 👥 Customer segmentation (Senior Citizen, Gender)
* 💳 Payment method distribution
* 🌐 Internet service analysis (DSL, Fiber, No Internet)
* 💰 Revenue and monthly charges trends
* 🎛️ Interactive slicers (Churn, Contract, Internet Service, Payment Method)
🔑 Key Insights
* Month-to-month contracts have the highest churn rate
* Lack of tech support and higher charges increase churn risk
* New customers are more likely to churn
* Long-term customers contribute more to revenue
🛠️ Tools & Technologies
* Power BI
* Data Visualization
* DAX (for measures like Churn Rate)
🚀 Conclusion
This dashboard helps businesses understand customer churn behavior and supports data-driven strategies to improve customer retention and revenue growth.
